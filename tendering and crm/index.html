How to deploy for a new client:
  1. Create a new Firebase project for the client
  2. Copy the HTML file, paste the Firebase config into the BRAND block at line ~29
  3. Change company name, colours & features in that BRAND block
  4. Log in as admin and use this panel to fine-tune — changes save to that client's Firestore
  5. Use "Export Current Config" below to snapshot the live config as JSON
